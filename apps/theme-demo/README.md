# Theme Demo

A demo site showcasing the Astro MD Theme.

This application demonstrates the default appearance and features of the theme package.

## Development

```bash
pnpm dev
```

The site will be available at `http://localhost:4321`

## Build

```bash
pnpm build
```

## RSS Feed

This site supports RSS subscription. Visit `/rss.xml` to get the latest posts with full content.

You can also find the RSS icon in your browser's address bar to subscribe.
