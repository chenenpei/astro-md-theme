# 欢迎使用 Astro MD Theme

这是一篇示例文章，你可以：

1. 修改这篇文章的内容
2. 在 `src/content/blog/` 目录下创建新的 Markdown 文件
3. 开始编写你自己的博客文章

## 开始使用

所有文章都存放在 `src/content/blog/` 目录下，使用 Markdown 格式编写。

支持标准的 Markdown 语法，包括：

- **粗体** 和 *斜体*
- 列表
- 代码块
- 链接和图片

祝你写作愉快！
